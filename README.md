# Automated Web Scraper

## Overview
A Python-based automated scraper designed to collect data daily, generate reports, and handle ban evasion.

## Hardware Environment
- Primary: Mac Pro 2025 (M4 Apple Silicon)
- Secondary: Mac Pro 2013 (Intel)

## Setup
1. Create a virtual environment: `python3 -m venv venv`
2. Activate: `source venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`

## Using the webscraper, step by step:

1. python scripts/daily_run.py
2. python src/reporter_html.py
